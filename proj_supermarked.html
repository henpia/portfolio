Supermarked

Simpel webapplikation til inventory af et supermarked.

Besøg websitet

Om dette projekt

Supermarked webapplikationen er ikke ment som en kundevendt indkøbsportal, men som en applikation, der kan opretholde et inventory i et supermarked.

Webapplikationen er en simpel applikation med en backend i form af en MSSQL database for persistense.

Teknologier

Der er anvendt følgende teknologier i udformningen af webapplikationen:

  • ASP.NET MVC
  • C#
  • Entity Framework / MSSQL Code First
  • HTML
  • CSS
  • Visual Studio
  • Git / GitHub
